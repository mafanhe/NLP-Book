# slides2.1

## 基本概念

* 自然语言指人类使用的语言，如汉语、英语等。

* 语言是思维的载体，是人际交流的工具。

* 语言的两种属性-文字和声音

* 人类历史上以语言文字形式记载和流传的知 识占知识总量的80%以上。

我们希望能够利用计算机来实现对语言的处理，并且对海量数据进行处理和有效利用
